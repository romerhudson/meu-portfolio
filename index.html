Sobre o Site

O layout do site foi criado no Adobe Illustrator e depois desenvolvido em HTML, CSS e Javascript usando linguagens puras.

O repositório do projeto está hospedado no github e a versão do meu currículo em PDF pode ser baixada clicando aqui.

Nos cards abaixo está um breve resumo de como essas linguagens foram utilizadas no projeto.

HTML

Partindo dos conceitos de boa semântica, acessibilidade e clareza do código, apliquei as melhores práticas em HTML para arquitetar o projeto.

CSS

Em conjunto com o HTML, o CSS deu "cara" à identidade visual do projeto. O modelo Flexbox foi empregado para organizar os elementos, enquanto as @media rules viabilizaram a responsividade do site.

JavaScript

O JavaScript foi fundamental para criar uma navbar responsiva com o burger icon e para criar o portfolio da seção 'Projetos'.
